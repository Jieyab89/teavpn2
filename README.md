
# TeaVPN2
TeaVPN2 is an open source VPN software written in C.
Supported platform is only Linux at the moment.


# Build
```
git clone https://github.com/TeaInside/teavpn2
cd teavpn2
make RELEASE_MODE=1 -j$(nproc)
```
